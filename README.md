## Getting Started

## Set up
## Install gems
`bundle install`

```

gem "rails", "~> 7.1.3"
gem "stimulus-rails"
gem "bootstrap", "~> 5.2"
gem "devise"
gem 'cloudinary', '~> 1.16.0'
gem 'faker'
```


## ENV Variables
### Create .env file
`touch .env`

## DB Setup
 ```
 rails db:create
 rails db:migrate
 rails db:seed
```
## Run a server
`rails s`

## Built With
### - Frontend: HTML, SCSS, Bootstrap, Javascript
### - Backend: Rails 7
### - Deployment: Heroku
### - PostgreSQL
### - Additional libraries: Stimulus
